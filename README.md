# NutriScan 

NutriScan is an AI-powered food recognition and nutrition analysis system.

## Features
- Food detection using YOLO
- Nutrition estimation
- Meal history tracking
- Calorie goal tracking
- Health score analysis
- Diet planner
- Water tracker
- Weight tracker

## Tech Stack
- Python
- YOLOv11 (Ultralytics)
- Streamlit
- SQLite
- Pandas

## Dataset
- IndianFoodNet
- OpenFoodDataset

## How to Run

Install dependencies:

pip install -r requirements.txt

Run the app:

streamlit run backend/nutriscan_app.py
